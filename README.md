# ATV4-Mobile

Nota sobre a API:
Para garantir a máxima fidelidade ao mockup fornecido, notei que a API sugerida (HG Brasil) não disponibiliza a previsão por hora (Hourly Forecast) no seu endpoint gratuito. Conforme as instruções do desafio ("Caso queira utilizar outra, escolha uma aberta e envie as intruções de uso"), optei por utilizar a Open-Meteo API.

Vantagens da Open-Meteo:

É 100% gratuita e de código aberto.

Não exige API Key (basta clonar o repositório e rodar o projeto, sem necessidade de configurações adicionais de ambiente por parte do avaliador).

Fornece arrays detalhados de previsão horária e diária.
